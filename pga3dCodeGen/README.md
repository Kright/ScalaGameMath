## module for code generation for pga3d

* [com.github.kright.pga3d.codegen](src/main/scala/com/github/kright/pga3d/codegen/scala) - code generation for Scala (result will be placed into ../pga3d)
* [com.github.kright.pga3d.codegencpp](src/main/scala/com/github/kright/pga3d/codegencpp) - code generation for C++ (result will be placed into ../cpp/pga3d)
